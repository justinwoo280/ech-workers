# P2-13: gRPC Conn.Close Cancel RecvMsg / gRPC Close 取消 RecvMsg

## 🟡 Severity / 优先级

**P2 — Medium (Goroutine Leak)** | Sprint 5 | 工作量: 0.25 天

## 📍 Location / 精确位置

- file:ewp-core/transport/grpc/conn.go `Close` 仅 `CloseSend`

## 🎯 Description / 漏洞描述与影响

不取消 RecvMsg,**潜在 goroutine 泄漏**。

## 💥 Reproduction / 复现条件

- 多次 Close 后 pprof 显示 RecvMsg 阻塞 goroutine 残留

## 🔧 Fix / 修复方案

加 ctx cancel:

```go
func (c *Conn) Close() error {
    c.cancel()  // 取消 ctx 让 RecvMsg 返回
    return c.stream.CloseSend()
}
```

## ✅ Acceptance Criteria / 验收标准

Close 后 RecvMsg goroutine 立即退出无 goroutine 泄漏

## 🧪 Verification / 验证方法

- **goroutine leak test**: 100 次 Close,断言 goroutine 数稳定

## 🔗 Dependencies / 依赖关系

- 与 `P1-10` (gRPC Pool 清理) 同模块

## ⚠️ Regression Risk / 回归风险

**低**
